HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the basic structure of a website using elements like headings, paragraphs, and links. Each element is defined using tags, which are enclosed in angle brackets (< and >). This simple example demonstrates the use of heading tags (H1 to H6) and a paragraph tag to structure content.
